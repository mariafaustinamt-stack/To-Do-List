# Basic To-Do List App

A simple and interactive To-Do List application built using HTML, CSS, and JavaScript to manage daily tasks efficiently.

## Features
- Add new tasks
- Delete tasks
- Interactive task management
- Clean and user-friendly interface
- Responsive layout

## Tech Stack
- HTML
- CSS
- JavaScript
